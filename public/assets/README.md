# Assets (gallery shell)

All image/video slots on the page are **blank placeholders**. Drop files here when you wire a real client:

| File name        | Type                | Where it shows up |
|------------------|---------------------|-------------------|
| `portrait.jpg`   | Image, 4:5          | Artist section portrait |
| `tv-loop.mp4`    | Video, 16:9 loop    | FrostbergTV screen |
| `battle-1.jpg` … | Images, 16:9        | Battle card thumbnails |

The gallery shell does not auto-load these — placeholders stay empty until you reconnect asset loading in `main.js`.
